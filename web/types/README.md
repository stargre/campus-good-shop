# types 目录说明

## 目录作用

此目录用于存储TypeScript类型定义文件。

## 文件说明

- **auto-imports.d.ts**：自动导入的类型定义（由unplugin-auto-import插件生成）
- **components.d.ts**：组件的类型定义（由自动组件注册插件生成）
- **env.d.ts**：环境变量的类型定义

## 注意事项

1. **自动生成**：部分文件由构建工具自动生成，不要手动修改
2. **类型定义**：可以在此目录添加自定义的类型定义文件
3. **版本控制**：建议将类型定义文件提交到Git

## 相关配置

TypeScript配置在`tsconfig.json`中，确保包含了此目录的类型定义。
