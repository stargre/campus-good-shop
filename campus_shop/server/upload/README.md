# upload 目录说明

## 目录作用

此目录用于存储用户上传的文件，包括图片、文档等。

## 子目录说明

- **avatar/**：用户头像文件
- **cover/**：商品封面图片
- **banner/**：轮播图文件
- **ad/**：广告图片
- **img/**：其他图片文件

## 注意事项

1. **文件管理**：此目录中的文件由系统自动管理，通常不需要手动修改
2. **备份**：生产环境建议定期备份此目录，防止文件丢失
3. **权限**：确保Web服务器有读写此目录的权限
4. **清理**：可以定期清理不再使用的文件，释放存储空间

## 配置说明

文件上传路径在`server/settings.py`中配置：
```python
MEDIA_ROOT = os.path.join(BASE_DIR, 'upload/')
MEDIA_URL = '/upload/'
```

## 安全建议

1. **文件类型验证**：后端应验证上传文件的类型和大小
2. **文件重命名**：上传的文件应重命名，避免文件名冲突和安全问题
3. **访问控制**：敏感文件应设置访问权限控制
